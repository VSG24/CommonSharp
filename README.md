# CommonSharp
CommonSharp - A collection of useful C# classes and static methods

v1.1.0 includes:

Static Methods:

`MD5()`
`SHA1()`
`RemoveSpecialCharacters()`
`ArrayRand()`

Classes:

`CryptoRandom` (What is it? Visit https://www.nuget.org/packages/CommonSharp/1.1.0)

**Install with nuget:**

`Install-Package CommonSharp`
